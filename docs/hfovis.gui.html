display_errors(messages: dict)

Displays error messages for fields that have validation issues by setting the style of the corresponding input widgets to indicate an error and setting their tooltips to the error messages.

apply_changes()

Applies changes made in the input widgets to the corresponding fields in the configurations. Validates the input and displays error messages if any issues are found.

reset_defaults()

Resets the input widgets to their default values as defined in the dataclass fields. Clears any error styles or tooltips.

disable_editing()

Disables editing of the input widgets based on the editable_after_start metadata of each field. If a field is not editable after the start, its input widget will be disabled.
